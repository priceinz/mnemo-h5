工作流名称：早9点推送昨日汇总
触发方式：定时触发
触发时间：每天 09:00

流程编排：
[定时触发: 每天09:00]
    ↓
[代码节点: 计算前一工作日日期]
    ↓
[HTTP请求: 获取前一日签到记录]
    ↓
[HTTP请求: 获取前一日日报记录]
    ↓
[代码节点: 整理汇总数据]
    ↓
[发消息节点: 推送汇总给你]


代码节点1（计算前一工作日）：
```javascript
// 获取当前日期
const now = new Date();
const dayOfWeek = now.getDay(); // 0=周日, 1=周一, ..., 6=周六

// 计算前一工作日
let targetDate = new Date(now);
if (dayOfWeek === 1) {
  // 周一 → 取上周五（往前推3天）
  targetDate.setDate(targetDate.getDate() - 3);
} else if (dayOfWeek === 0) {
  // 周日 → 取上周五（往前推2天）
  targetDate.setDate(targetDate.getDate() - 2);
} else {
  // 周二到周六 → 取前一天
  targetDate.setDate(targetDate.getDate() - 1);
}

// 格式化日期
const dateStr = targetDate.toLocaleDateString('zh-CN', {
  month: 'long',
  day: 'numeric',
  weekday: 'short'
});

// 生成时间戳（毫秒）
const startOfDay = new Date(targetDate);
startOfDay.setHours(0, 0, 0, 0);
const startTime = startOfDay.getTime();

const endOfDay = new Date(targetDate);
endOfDay.setHours(23, 59, 59, 999);
const endTime = endOfDay.getTime();

return {
  dateStr: dateStr,
  startTime: startTime,
  endTime: endTime,
  isoDate: targetDate.toISOString().split('T')[0]
};
```

HTTP请求1（获取签到数据）：
```
请求方式：POST
请求地址：https://oapi.dingtalk.com/topapi/checkin/record/get
Body(JSON)：
{
  "userid_list": "${员工user_id列表}",
  "start_time": "${startTime}",
  "end_time": "${endTime}",
  "cursor": 0,
  "size": 100
}
```

HTTP请求2（获取日报数据）：
```
请求方式：POST
请求地址：https://oapi.dingtalk.com/topapi/report/list
Body(JSON)：
{
  "start_time": "${startTime}",
  "end_time": "${endTime}",
  "template_name": "日报",
  "cursor": 0,
  "size": 100
}
```

代码节点2（整理汇总）：
```javascript
// 员工配置（替换为真实的user_id和姓名）
const employees = [
  {name: "张三", userid: "实际user_id", group: "工程审计"},
  {name: "李四", userid: "实际user_id", group: "供应监察"},
  {name: "王五", userid: "实际user_id", group: "供应监察"},
  {name: "赵六", userid: "实际user_id", group: "内部审计"},
  {name: "钱七", userid: "实际user_id", group: "内部审计"},
  {name: "孙八", userid: "实际user_id", group: "内部审计"},
  {name: "周九", userid: "实际user_id", group: "内部审计"},
];

const targetDate = input.dateStr;

// 处理签到数据
const checkinList = input.checkinResult?.result?.page_list || [];
const checkinMap = {};
checkinList.forEach(r => {
  if (!checkinMap[r.userid]) checkinMap[r.userid] = [];
  checkinMap[r.userid].push(r);
});

// 处理日报数据
const reportList = input.reportResult?.result?.data_list || [];
const reportMap = {};
reportList.forEach(r => {
  reportMap[r.creator_id] = r;
});

// 按组整理
const groups = {};
let checkinCount = 0;
let reportCount = 0;

employees.forEach(emp => {
  if (!groups[emp.group]) groups[emp.group] = [];

  // 签到信息
  const checkins = checkinMap[emp.userid] || [];
  const hasCheckin = checkins.length > 0;
  if (hasCheckin) checkinCount++;

  let checkinInfo = "";
  if (hasCheckin) {
    const first = checkins[0];
    const time = new Date(first.checkin_time).toLocaleTimeString('zh-CN', {
      hour: '2-digit', minute: '2-digit'
    });
    checkinInfo = `${first.place || '未知地点'}（${time}）`;
  }

  // 日报信息
  const report = reportMap[emp.userid];
  const hasReport = !!report;
  if (hasReport) reportCount++;

  let summary = "";
  if (report && report.contents && report.contents.length > 0) {
    const val = report.contents[0].value || "";
    summary = val.length > 60 ? val.substring(0, 60) + "..." : val;
  }

  groups[emp.group].push({
    name: emp.name,
    checkin: hasCheckin,
    checkinInfo: checkinInfo,
    report: hasReport,
    summary: summary
  });
});

// 生成汇总文本
let text = `📋 昨日工作汇总 — ${targetDate}\n\n`;

for (const [group, members] of Object.entries(groups)) {
  text += `📁 ${group}\n`;
  members.forEach(m => {
    text += `  ${m.name}：\n`;
    text += `    ${m.checkin ? '✅' : '⚠️'} 签到 ${m.checkin ? '| ' + m.checkinInfo : '未签到'}\n`;
    text += `    ${m.report ? '✅' : '❌'} 日报 ${m.report ? '| ' + m.summary : '未提交'}\n`;
  });
  text += "\n";
}

text += `📊 统计：签到 ${checkinCount}/7 | 日报 ${reportCount}/7`;

// 异常提醒
const absent = employees.filter(e => !checkinMap[e.userid] || checkinMap[e.userid].length === 0);
const missingReport = employees.filter(e => !reportMap[e.userid]);

if (absent.length > 0 || missingReport.length > 0) {
  text += "\n\n🚨 异常情况：\n";
  if (absent.length > 0) {
    text += `  ⚠️ 未签到：${absent.map(e => e.name).join('、')}\n`;
  }
  if (missingReport.length > 0) {
    text += `  ❌ 未交日报：${missingReport.map(e => e.name).join('、')}`;
  }
}

return { summaryText: text };
```

发消息节点：
- 接收人：你的user_id（或选择"自己"）
- 消息内容：${summaryText}

注意：
1. 需要替换员工列表中的"实际user_id"为真实的钉钉user_id
2. 需要在HTTP请求中添加access_token参数
3. 需要先创建钉钉开放平台应用获取AppKey和AppSecret
