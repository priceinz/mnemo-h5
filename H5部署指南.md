# MNEMO H5 部署指南 — 10 分钟上线

---

## 第一步：安装 Node.js（如果没装的话）

打开 https://nodejs.org 下载 LTS 版本，安装。

确认安装成功：
```powershell
node --version
npm --version
```

都显示版本号就OK。

---

## 第二步：把 h5-deploy 文件夹放到桌面

把下载的 `h5-deploy` 整个文件夹放到：
```
C:\Users\Zane\Desktop\vibecoding\脑子ai助手\
```

---

## 第三步：安装依赖 + 本地预览

打开 PowerShell，输入：

```powershell
cd "$HOME\Desktop\vibecoding\脑子ai助手\h5-deploy"
npm install
npm run dev
```

等几秒，终端会显示类似：
```
  Local:   http://localhost:5173/
```

浏览器打开这个地址，应该能看到完整的 MNEMO 应用。
手机和电脑连同一个 WiFi，手机浏览器打开 Network 那行地址也能看。

**确认效果没问题后，继续下一步。**

按 Ctrl+C 停掉本地服务。

---

## 第四步：部署到 Vercel（免费，永久在线）

### 4.1 注册 Vercel

1. 打开 https://vercel.com
2. 点 Sign Up → Continue with GitHub
3. 如果没有 GitHub 账号，先去 https://github.com 注册一个（免费）
4. 授权登录

### 4.2 安装 Vercel CLI

```powershell
npm install -g vercel
```

### 4.3 登录

```powershell
vercel login
```

选 GitHub，浏览器会弹出授权页面，点确认。

### 4.4 部署！

确保你在 h5-deploy 目录下：

```powershell
cd "$HOME\Desktop\vibecoding\脑子ai助手\h5-deploy"
vercel
```

它会问你几个问题，这样回答：

```
? Set up and deploy? → Y（回车）
? Which scope? → 选你的账号（回车）
? Link to existing project? → N（回车）
? What's your project's name? → mnemo（回车）
? In which directory is your code located? → ./（回车）
? Want to modify these settings? → N（回车）
```

等 1-2 分钟，部署完成后会显示：

```
✅ Production: https://mnemo-xxxxx.vercel.app
```

**这个网址就是你的 App！** 手机微信里打开它，效果和你在 Claude 里看到的完全一样。

### 4.5 自定义域名（可选）

如果你有自己的域名，在 Vercel 后台 Settings → Domains 添加。
如果没有，直接用 vercel.app 的免费域名。

---

## 第五步：添加到手机桌面

### iPhone
1. 微信里打开你的网址
2. 点右上角 ··· → 「在 Safari 中打开」
3. Safari 里点分享按钮 → 「添加到主屏幕」
4. 图标和名称会是 MNEMO，像原生 App 一样

### Android
1. 微信里打开网址
2. 点右上角 ··· → 「在浏览器中打开」
3. 浏览器菜单 → 「添加到桌面」

---

## 第六步：后续更新

以后修改了代码，只需要在 h5-deploy 目录下运行：

```powershell
cd "$HOME\Desktop\vibecoding\脑子ai助手\h5-deploy"
vercel --prod
```

30 秒自动更新上线，网址不变。

## 常见问题

### npm install 报错
```powershell
# 清除缓存重试
npm cache clean --force
npm install
```

### vercel 命令找不到
```powershell
# 重新安装
npm install -g vercel
# 关掉 PowerShell 重新打开再试
```

### 手机上看到的是桌面版布局
网页会自动适配手机宽度。如果看起来不对，在微信里打开（不是在电脑浏览器）。

### 想要 HTTPS
Vercel 默认就是 HTTPS，不需要额外操作。
